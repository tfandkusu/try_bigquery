# try_bigquery

PythonでのBigQueryからのデータの取り出しとBigQueryテーブルへの追記のやり方を確認する

# セットアップ方法

M1 Macの場合

```sh
pyenv install
pip install poetry
export GRPC_PYTHON_BUILD_SYSTEM_OPENSSL=1
export GRPC_PYTHON_BUILD_SYSTEM_ZLIB=1
poetry install
```
